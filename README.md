# AI Maze Runner & Pathfinding Algorithm Visualizer

## Overview

AI Maze Runner & Pathfinding Algorithm Visualizer is an interactive Python application that demonstrates and compares popular pathfinding algorithms on randomly generated mazes.

The project allows users to generate new mazes and visualize how different algorithms search for a path from the start node to the goal node.

---

## Features

* Random Maze Generation
* Breadth First Search (BFS)
* Depth First Search (DFS)
* A* Search Algorithm
* Interactive Tkinter GUI
* New Maze Generation
* Path Length Analysis
* Execution Time Comparison

## Algorithms Implemented

### Breadth First Search (BFS)

Explores nodes level by level and guarantees the shortest path in an unweighted maze.

### Depth First Search (DFS)

Explores one path deeply before backtracking.

### A* Search

Uses a heuristic (Manhattan Distance) to efficiently find the shortest path.
